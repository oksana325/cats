Идеи для клички

  1. В зависимости от окраса.
    • Для кошки: Белочка, Арабика, Пинки, Дымка, Санни, Тень, Бланка, Лили, Фокси, Блэки, Полоска, Брауни, Злата.
    • Для кота: Дымок, Уголек, Апельсин, Лисенок, Персик, Грей, Сумрак, Гризли, Лотос, Снежок, Табби, Абрикос, Сноу, Найт, Мандарин.
  2. В зависимости от породы и страны, где она была выведена.
    • Для кошки: Багира, Айрис, Диана, Мими, Сакура, Индия, Тиффани, Скарлетт, Пикси, Лиззи, Алиса, Лило, Ямайка, Пэрис, Мальта, Аляска.
    • Для кота: Сиам, Оскар, Феликс, Гарри, Байрон, Чарли, Симба, Тао, Абу, Ватсон, Енисей, Измир, Кайзер, Васаби, Байкал.
  3. В зависимости от его внешности, поведения и характера.
    • Для кошки: Соня, Клеопатра, Цунами, Забияка, Матильда, Кнопка, Масяня, Царапка, Серсея, Ворсинка, Амели, Наоми, Маркиза, Изольда.
    • Для кота: Гарфилд, Том, Гудвин, Рокки, Ленивец, Пушок, Спорти, Бегемот, Пират, Гудини, Зорро, Саймон, Альбус, Базилио, Леопольд, Нарцисс, Атос, Каспер, Валли.
